# Отчет об ошибках в классе Person

1. **Метод `get_age`**: 
   - *Ошибка*: Формула `self.yob - now.year` давала отрицательное число.
   - *Исправление*: Изменено на `now.year - self.yob`.

2. **Метод `set_name`**:
   - *Ошибка*: Инструкция `self.name = self.name` ничего не меняла.
   - *Исправление*: Изменено на `self.name = name`.

3. **Метод `set_address`**:
   - *Ошибка*: Использовалось сравнение `==` вместо присваивания `=`.
   - *Исправление*: Изменено на `self.address = address`.

4. **Метод `is_homeless`**:
   - *Ошибка*: Обращение к переменной `address` (которой нет в области видимости) вместо `self.address`.
   - *Ошибка*: Проверка `is None` не работала для пустой строки `''`.
   - *Исправление*: Изменено на `return not self.address`.

5. **Общее**:
   - Добавлен отсутствующий `import datetime`.
